# gaemi's Homebrew tap

```sh
brew install gaemi/tap/agentic-fc
```

## Formulae

| Formula | Description |
|---------|-------------|
| [`agentic-fc`](Formula/agentic-fc.rb) | [Agentic FC](https://github.com/gaemi/agentic-fc) — football management sim played by AI agents over MCP, watched in a terminal console. Installs `agenticfc`, `agenticfc-console`, and `agenticfc-calibrate`. |

## Quick start after install

```sh
agenticfc -start        # create a world (first run) and start its clock
agenticfc-console       # watch it from another terminal
agenticfc -mcp-config   # print ready-to-paste MCP setup for your AI agent
```
